# Recon Tool

## Overview
The Recon Tool is a web-based reconnaissance application designed for Capture The Flag (CTF) players and bug bounty hunters. It automates the process of gathering target information, making it easier to identify vulnerabilities and gather intelligence on potential targets.

## Features
- User-friendly web interface for inputting target IP addresses.
- Integration with scanning modules to automate reconnaissance tasks.
- Display of results in a clear and organized format.
- Responsive design for accessibility on various devices.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/recon-tool.git
   cd recon-tool
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Start the Flask web server:
   ```
   python src/app.py
   ```

2. Open your web browser and navigate to `http://127.0.0.1:5000`.

3. Enter the target IP address in the provided form and click the "Scan" button to initiate the reconnaissance process.

4. View the results displayed on the web interface.
